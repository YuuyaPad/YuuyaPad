# Contribution Guidelines
Contributions to YuuyaPad are welcome, provided they meet the basic guidelines described below.
* Don't use vague commit messages.
* Don't try to add completed [future goals](https://github.com/YuuyaPad/YuuyaPad#future-goals).
* Don't add things that are difficult to implement.
* If you don't know how to code or aren't good at C#, it's fine to use a generator like ChatGPT.
* Commenting code is optional, but it's recommended to make it easier for everyone to understand.
